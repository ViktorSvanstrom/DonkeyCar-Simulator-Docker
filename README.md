# DonkeyCar Simulator

## Requirements by Platform

### Windows:
- Docker Desktop
- X11 server (VcXsrv eller Xming) för att visa GUI:n

### macOS:
- Docker Desktop
- XQuartz för X11-stöd

### Linux:
- Docker Engine
- X11 forwarding (vanligtvis redan konfigurerat)

## How to Run

```bash
docker-compose up --build
```
